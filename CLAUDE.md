# NORTHEPTION SNS半自動投稿システム

## プロジェクト概要
NORTHEPTIONのX（Twitter）投稿を半自動化するDiscord Bot。
担当者がDiscordのプライベートチャンネルに何かを書く → Claudeが3パターンの文案を生成 → 担当者がコピペしてXに投稿。

## 重要なルール
- X APIは使用しない（手動コピペ投稿）
- 発信の最終判断は必ず人間が行う（全自動投稿禁止）
- ブランド安全性最優先
- 黄色いハート（💛）は絶対に使用しない
- #NthWIN #SF6 は必ず末尾に入れる
- 担当者が何を書いても必ず3パターン返す（1対3を厳守）
